# 🚀 Parth Vasave - Developer Portfolio

This is my personal developer portfolio website, built using **React** and deployed on **GitHub Pages**. It showcases my work, skills, and contact information in a sleek, modern interface.

🔗 **Live Site:** [https://Parth-Vasave.github.io/Portfolio](https://Parth-Vasave.github.io/Portfolio)

---

## 📁 Features

- ✨ Smooth animations and interactive UI
- 🧠 About me section
- 🛠 Projects with visual previews and tech stack
- 💡 Skills illustrated using custom icons
- 📫 Contact section with direct reach options

---

## 🛠 Built With

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [gh-pages](https://www.npmjs.com/package/gh-pages)
- [CSS Animations](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Animations)
